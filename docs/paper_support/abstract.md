# Abstract

Electrical impedance analysis and Nyquist visualization are widely used across research and engineering domains, but accessible software workflows are often fragmented or difficult to reproduce. SynAptIp LCR Control V3 addresses this gap by providing a structured desktop platform for impedance-oriented data processing and Nyquist export workflows.

The software implements a computational transformation method that maps magnitude-phase impedance inputs into Nyquist coordinates, including `z_real`, `z_imag`, and `minus_z_imag`, and supports standardized CSV/JPG outputs for downstream analysis and reporting.

Using illustrative example datasets included in the repository, the workflow demonstrates reproducible generation of transformed Nyquist data and comparison plots under a documented and offline-capable process.

This positioning supports practical use in research, teaching, and engineering environments where transparent computational steps, traceable outputs, and reproducibility-focused documentation are required.
