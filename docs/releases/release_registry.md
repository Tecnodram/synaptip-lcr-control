## Major Capabilities (V3.5.0)

- Stable V3 instrument control compatibility retained.
- Analysis and Insights workflow for EIS post-processing.
- Nyquist, Bode, and derived-domain plot generation from cleaned data.
- Auditable cleaning pipeline with removed-point traceability.
- Heuristic interpretation report output (text and markdown).
- Metadata-rich export structure for reproducibility and validation.
- Frozen executable packaging with startup robustness logging.

## Known Limitations

- Equivalent-circuit fitting is not included in V3.5.0.
- Kramers-Kronig validation is not included in V3.5.0.
- Interpretation remains heuristic and non-definitive by design.
- Windows executable is the primary packaged distribution target.
